# Release v0.1.0 - 2026-05-18

## Summary

Initial scaffold for `key-vault`. Repository created on GitHub, standards in place, CI configured, ready for Phase 0.2.0 implementation work.

This crate provides enterprise-grade in-memory key management for the Hive DB stack and the wider Rust ecosystem. It uses defense-in-depth (memory page locking, scattered storage with distortion, zero-on-drop, constant-time operations, pluggable acquisition) to make cryptographic key material hard to extract via memory analysis.
